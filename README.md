支持广告商发布 / 分类广告，提供精准投放 API，对接跨站用户行为追踪，是匿名广告投放生态的核心系统

## 技术选型
- 后端：Spring Boot 2.7.x + MyBatis Plus（JDK 1.8，部署于Tomcat 9.0）
- 前端：Vue.js 2.x + Element UI
- 数据库：MySQL 8.0 + Redis 6.0
- 版本控制：Git + GitHub（单仓库多模块管理）
